# Flutter Assignment: Smart Services Marketplace App

### Duration

**90 Minutes (1.5 Hours)**

## Objective

Build a **Smart Services Marketplace App** where users can browse services, view service provider details, and contact providers directly from the app.

This assignment will help you practice:

* Flutter UI Design
* Navigation
* ListView/GridView
* Data Passing Between Screens
* User Interaction
* URL Launcher (Call Feature)
* Project Structure

---

# Features to Implement

## 1. Dashboard Screen

Create a Dashboard Screen with:

### AppBar

**Title:** Smart Services Marketplace

### Navigation Drawer

Include:

* Home
* My Profile
* About Us
* Logout

### Service Categories

Display at least **6 services** using ListView or GridView.

Example Services:

* Electrician
* Plumber
* Carpenter
* Painter
* AC Repair
* Cleaning Service

Each service card should display:

* Service Icon
* Service Name
* Short Description

---

## 2. Service Provider List Screen

When a user taps a service category, navigate to a screen showing available service providers.

Display at least **3 providers per service**.

Each provider card should contain:

* Profile Image/Icon
* Provider Name
* Experience (Years)
* Rating (⭐)
* Contact Number

Example:

```text
Rahul Electric Works

Experience: 5 Years
Rating: ⭐ 4.5
Phone: 9876543210
```

---

## 3. Provider Details Screen

When a provider is selected, show:

### Provider Information

* Name
* Service Type
* Experience
* Rating
* Contact Number
* Address
* About Provider

### Example

```text
Name: Rahul Electric Works

Service: Electrician

Experience: 5 Years

Rating: ⭐ 4.5

Address:
Ahmedabad, Gujarat

About:
Professional electrical installation and repair specialist.
```

---

## 4. Call Service Provider Feature ⭐

Add a **Call Now** button.

When the button is clicked:

```dart
launchUrl(Uri.parse('tel:9876543210'));
```

The device should open the dialer with the provider's phone number.

### Package

Use:

```yaml
url_launcher: ^6.2.5
```

---

## 5. WhatsApp Contact Feature ⭐

Add a **WhatsApp** button.

Example:

```dart
https://wa.me/919876543210
```

When clicked, WhatsApp should open with the provider's number.

---

## 6. Search Services Feature

Add a TextField on the Dashboard.

Users should be able to search services by name.

Example:

```text
Search: Plumber
```

Only matching services should appear.

---
